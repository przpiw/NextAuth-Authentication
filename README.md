### NextAuth v4

# Example of NextAuth with custom login form & credential provider to handle signin with own credential verification.

# A project bolilerplate to save up time on building authentication functionality from scrach.

# Own database connection or api/endpoint required for credential verification

Benefits of using NextAuth v4

- Encrypted JWT(JWE)
- Sessions supported
- Secure by default
- Bring your own database friendly
- Cross Site Request Forgery Tokens
- easy to handle errors
- Restrictive cookie policy
